A professional portfolio website for Gourav Ambawat, Pharmacist & Healthcare Professional.

---

## 🚀 Deploy in One Click

### Option 1 — Netlify (Easiest, Recommended)

1. Go to [netlify.com](https://netlify.com) and sign up free
2. Click **"Add new site" → "Deploy manually"**
3. Drag and drop this entire folder onto the page
4. ✅ Your site is live instantly at a `*.netlify.app` URL

**Or connect via GitHub:**
1. Push this folder to a GitHub repo
2. On Netlify: **"Add new site" → "Import from Git"**
3. Select your repo — Netlify auto-detects `netlify.toml`
4. Click **Deploy** — done!

---

### Option 2 — GitHub Pages (Free)

1. Create a new repository on [github.com](https://github.com)
2. Upload all files in this folder to the repo
3. Go to **Settings → Pages**
4. Set source to **"GitHub Actions"**
5. The included workflow (`.github/workflows/deploy.yml`) will auto-deploy
6. ✅ Live at `https://yourusername.github.io/repo-name`

---

### Option 3 — Vercel (Fast)

1. Go to [vercel.com](https://vercel.com) and sign up free
2. Click **"Add New Project"**
3. Import your GitHub repo (or use Vercel CLI: `npx vercel`)
4. ✅ Live at a `*.vercel.app` URL in seconds

---

### Option 4 — Manual Upload (Any Web Host)

Just upload `index.html` to any web hosting service:
- Hostinger, GoDaddy, cPanel, InfinityFree, etc.
- Upload via File Manager or FTP
- No build step needed — it's pure HTML
